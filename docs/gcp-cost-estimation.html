Tally CRM — GCP Infrastructure & Cost Estimation

Detailed analysis covering all environments, services, and commercial modelling

Prepared: March 2026 Currency: AUD (with USD reference) Users: ~30 Transaction Volume: Low–Mid

Key Assumptions

  • ~30 users across the platform
  • Low-to-mid transaction volume — not a high-frequency trading or analytics platform
  • 4 environments: Production, Pre-prod (UAT), Test/QA, Development
  • Current modules: Cases, Pipeline, Customer Management, Communications, Contracts, Dashboard, Performance, Settings
  • Sales module to be added later — estimated at ~50% the size of Case Management
  • Single GCP region (e.g. australia-southeast1 for Sydney)
  • GitLab for source control and CI/CD
  • All prices are on-demand unless noted — committed-use discounts shown separately

Final Recommendation

Go with Cloud SQL PostgreSQL

  • Saves $30,000–$42,000 AUD/year vs AlloyDB across all 4 environments
  • 30 users with low-to-mid transactions is well within Cloud SQL's capacity
  • 99.95% SLA with HA — more than adequate for a business CRM
  • Straightforward migration path to AlloyDB if you outgrow it
  • Budget $620–$1,020 AUD/month for Phase 1 (all environments, all services)
  • Budget $710–$1,270 AUD/month after the Sales module is added
  • With a 1-year commitment, annual cost drops to $5,840–$9,608 AUD
